FLIMS Aviation records, reports, and logistics

Project Description

(no demo available)

The Flightline Information Management System (FLIMS) is an aviation record management system designed to streamline the entire flight operations process. The scope of this project ranges from flight data entry, to government reporting, to manpower tracking. The end result is a system that makes any flight operation more efficient and accurate.
